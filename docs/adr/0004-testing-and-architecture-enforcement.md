# ADR 0004: Тестирование и архитектурный контроль

## Решение

Командные правила проверяются тестами, а не только code review.

## Обязательные тесты

- Тесты use cases покрывают оркестрацию через application-порты.
- Тесты fan-out покрывают лимиты на один запрос, проброс ошибок, освобождение capacity и обработку interrupt.
- Примерные тесты адаптеров покрывают входящий REST, входящий gRPC, исходящий REST и исходящий gRPC.
- ArchUnit и Spring Modulith тесты проверяют направление зависимостей и владение concurrency-механизмами.
